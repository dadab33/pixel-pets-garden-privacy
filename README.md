# pixel-pets-garden-privacy

Politique de confidentialité — Pixel Pets Garden

Données collectées :
- Identifiant publicitaire Android (AAID) par Google AdMob pour l'affichage des publicités.
- Données de transaction anonymisées par Google Play Billing.

Données NON collectées :
- Aucune donnée personnelle (nom, email, localisation GPS).
- Les pas enregistrés par le podomètre restent locaux sur l'appareil et ne sont jamais transmis.

Partage avec des tiers :
- Google AdMob (publicités) : https://policies.google.com/privacy
- Google Play Billing (achats) : https://play.google.com/about/play-terms

Public cible : tous âges (3+).
Conformité COPPA : l'app ne collecte aucune donnée des enfants.
